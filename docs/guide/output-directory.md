# Директория вывода

По умолчанию, `tsdown` собирает ваш код в директорию `dist`, который находится в текущем рабочем каталоге.

Если вы хотите изменить директорию вывода, используйте опцию `--out-dir` (или `-d`):

```bash
tsdown -d ./custom-output
```

## Пример

```bash
# Поведение по умолчанию: вывод в ./dist
tsdown

# Пользовательская директория вывода: вывод в ./build
tsdown -d ./build
```

> [!NOTE]
> Указанная директория вывода будет создана, если она ещё не существует. Убедитесь, что путь к директории соответствует структуре вашего проекта, чтобы избежать нежелательной перезаписи файлов.
