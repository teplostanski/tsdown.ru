# Поддержка CSS

Поддержка CSS в `tsdown` находится на очень ранней экспериментальной стадии. Хотя базовые возможности уже доступны, API и поведение могут измениться в будущих версиях.

> [!WARNING] Экспериментальная функция
> Поддержка CSS крайне экспериментальна. Тщательно тестируйте её и сообщайте о любых проблемах. API и поведение будут меняться по мере развития функции.

## Настройки

### Отключение разделения CSS

По умолчанию CSS может разбиваться на несколько файлов в зависимости от точек входа. Чтобы отключить разделение CSS и сгенерировать единый файл стилей, установите параметр `css.splitting` в `false` в конфигурации:

```ts
export default defineConfig({
  css: {
    splitting: false,
  },
})
```

### Указание имени выходного CSS-файла

Имя объединённого CSS-файла можно настроить с помощью опции `css.fileName`:

```ts
export default defineConfig({
  css: {
    fileName: 'my-library.css',
  },
})
```

В результате весь CSS будет выведен в файл `my-library.css` в директории вывода.
